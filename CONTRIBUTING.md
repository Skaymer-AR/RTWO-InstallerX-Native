# Contributing

Contributions should arrive through forks and pull requests. Direct edits to the
default branch are not part of the intended workflow.

Useful contributions:

- support for another RTWO Android 16 build, with exact hashes and logs;
- safer compatibility aliases for installer/uninstaller activities;
- recovery diagnostics;
- reproducible build improvements;
- documentation corrections.

Do not submit stock Motorola framework binaries, private logs, signing keys or
personal package databases.
